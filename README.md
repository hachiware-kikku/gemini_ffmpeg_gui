# gemini_ffmpeg_gui
Google Geminiに作成してもらうffmpegのGUI

## プロジェクトの方針

- ビルドツール : gradle
- 利用できるOSSライブラリのライセンス : Apache, MIT, BSD, EPL
- GUIはSWTで構築
